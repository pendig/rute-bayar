# Support

Rute Bayar is an open source alpha project. Support is community/best-effort.

## Where to Ask

- Use GitHub Issues for reproducible bugs and scoped feature requests.
- Use pull requests for proposed fixes.
- Include provider, environment, command, and sanitized output when asking for help.

## Before Opening an Issue

Please check:

- [README.md](./README.md)
- [docs/end-to-end-smoke.md](./docs/end-to-end-smoke.md)
- [docs/operations-runbook.md](./docs/operations-runbook.md)
- Existing issues and pull requests.

## What to Include

For bugs:

- Rute Bayar version or commit SHA.
- Operating system and Go version.
- Provider and environment, such as Midtrans sandbox or Xendit sandbox.
- CLI command used.
- Sanitized logs or raw JSON snippets.
- Expected result and actual result.

Do not include credentials, webhook tokens, customer personal data, or full raw payloads with secrets.
